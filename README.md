# 简介

1、遵守软件开发友好原则 ，及时 释放 CPU 更省电。

2、秉承昕浏览器开发原则，简约而不失强悍。

3、采用最新 AgentWeb 支持。更快更安全!

## 特色功能

1、多种搜索引擎并存，快速切换

2、支持 Google 绿色上网

3、支持多种设计分享和腾讯收藏

4、支持手势操作

5、支持语音搜索

## 核心理念

> “专注产品质量优化！不打扰，不去故意很懂你。”

## 最新功能

- 多种社交分享，和朋友一起玩！
